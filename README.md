# se-takehome
## Create search a small frontend application showing a usable search interface complete with a bar, search results page, and filters utilizing the data of your choice.

Hello, Algolia!

First, thank you so much for the opportunity to participate in this exercise. Having this hands-on exposure to the product dashboard and developer libraries deepens my understanding of the technologies that will be used in the Solution Engineering role at Algolia.

My approach was to begin at the Algolia Academy website and follow some of the technical introductory courses for the Algolia Dashboard and settings, as well as the options for building and connecting a user interface. Afterward, I reviewed the documentation for the API and UI Libraries to acclimate myself to the format and resource navigation.

For this assignment, I selected the `ecommerce/records.json` (Best Buy) dataset from the supplied options in the Algolia GitHub dataset collection. It seemed to have attributes that matched my ecommerce and product discovery experience. I was amazed at how intuitive the admin dashboard interface is for uploading and managing the dataset(s). Stepping through the Configurations panel for the indices, I explored the opportunities to optimize (or ruin) relevancy. I found myself experimenting with different adjustments and observing the impact each had on rankings.

For the interface build, I elected to implement `instantsearch.js` using Vanilla.js, CSS, and HTML in a single-page experience. This allows focused testing by the end user, as well as immediate review of the complete local source code without tracing abstracted files or deciphering minification from modern framework processing. While working through the instantsearch.js documentation's table of contents, I found it almost addicting as I internally justified the time to implement "just one more" widget as a "must-have" in the UI.

Overall, I am very pleased with the experience and the result. However, I only scratched the surface at Algolia. There is much to explore and get creative with, including more complex widgets, a deeper understanding (and impact) of the relevancy settings, Agent Studio, personalization, data transformations, and more.

I sincerely hope you find the assignment satisfactory, and I want to thank you once again for this opportunity.
